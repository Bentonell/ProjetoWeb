## Descrição
Este projeto consiste num website temático de Natal, desenvolvido com **HTML5** e **CSS3**.  
O site é **100% funcional**, responsivo e segue os requisitos da disciplina, incluindo:

- **3 páginas HTML obrigatórias:**
  - `index.html` (Home / Landing page)
  - `sobre.html` (Sobre / Serviços / Portefólio)
  - `contactos.html` (Contactos, com formulário visual)
- **Menu de navegação consistente** entre todas as páginas
- **Header e Footer** em todas as páginas
- **Secções principais** com conteúdo realista (texto Lorem Ipsum incluído)
- **Links internos** entre páginas e **links externos** (ex.: redes sociais)
- **Imagens** com alt-text e favicon incluído
- **Layout responsivo** para dispositivos Mobile, Tablet e Desktop
- Uso de **CSS Grid** para layout principal e **Flexbox** como complemento
